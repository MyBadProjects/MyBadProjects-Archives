# UK-Phone-Number-Generator
This is desinged to make fake UK Phone Numbers.
There are 3 modes.

## How to implement

To easily implemnt it do:
```javascript
const ukpng = require('UK-Phone-Number-Generator')
```

To use it you can do:
```javascript
const ukpng = require('UK-Phone-Number-Generator')

ukpng.GenerateNumber(3)
```

This is has 3 commands built in.

#### Commands Info
2 - Help
1 - Generation
